# Marsic AI Lab — AI Web App Builder Portfolio

Portfolio web app modern untuk menampilkan kemampuan membangun website dan prototype aplikasi menggunakan workflow AI, prompt engineering, UI direction, dan automation-assisted development.

Project ini dibuat dengan Blazor Web App dan berisi landing page portfolio, demo aplikasi manajemen studio musik, serta admin dashboard interaktif.

## ✨ Highlights

- Modern AI creator landing page
- Interactive studio management demo
- Admin dashboard simulation
- Booking management UI
- Invoice preview and print mode
- Customer and finance dashboard
- Responsive layout
- Neon glassmorphism visual style
- External animation libraries integration

## 🧠 Positioning

Saya membangun web app dengan pendekatan AI-assisted development.

Fokus utama project ini bukan hanya menulis kode, tetapi mengubah ide menjadi prototype digital yang bisa dilihat, diklik, diuji, dan dikembangkan lebih lanjut.

Workflow yang digunakan:

- Prompt direction
- AI-assisted code generation
- UI/UX polishing
- Feature testing
- Iterative bug fixing
- Product presentation
