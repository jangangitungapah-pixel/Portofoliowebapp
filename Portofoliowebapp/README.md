# Marsic AI Lab — AI Web App Builder Portfolio

Modern AI-assisted web app portfolio built with Blazor Web App.

This project showcases a personal portfolio landing page, an interactive studio management app demo, and an admin dashboard simulation. It is designed to present how ideas can be transformed into clickable digital prototypes using AI-assisted development, prompt engineering, UI direction, and iterative polishing.

---

## ✨ Project Highlights

- Modern AI creator landing page
- Interactive Studio Management App demo
- Admin Dashboard simulation
- Booking management flow
- Search and filter booking data
- Booking status update
- Confirmation modal for destructive actions
- Toast notification system
- Invoice preview
- Print-ready invoice layout
- Copy invoice summary action
- Responsive neon glassmorphism UI
- External animation and interaction libraries

## 📦 How to Run Locally

Clone the repository:

```bash
git clone https://github.com/jangangitungapah-pixel/Portofoliowebapp.git
```

Open the project in Visual Studio Community.

Then run:

```txt
Ctrl + F5
```

Or run from terminal:

```bash
dotnet run
```

After running, open the local URL shown by the app.

Example routes:

```txt
/
/studio-demo
/studio-admin
```
